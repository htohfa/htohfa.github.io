What I Do

My research sits at the intersection of computational physics, machine learning, and observational astronomy

Machine Learning for Astronomy

Building neural networks for telescope optimization, developing compression algorithms that preserve scientific information, and applying deep learning to cosmological parameter estimation.

Advanced Statistical Methods

Using wavelet scattering transforms to extract non-Gaussian information from astronomical spectra, achieving order-of-magnitude improvements in cosmological constraints.

Large-Scale Simulations

Enhancing simulation frameworks like MP-Gadget, running cosmological simulations on HPC clusters, and developing new physics modules for multi-component systems.

Fundamental Physics Tests

Using CMB data and quasar spectra to test whether fundamental constants have evolved over cosmic time, constraining models beyond the Standard Model.

Software Development

Writing efficient Python and Julia code, developing analysis pipelines for large datasets, and creating tools that make complex astrophysics more accessible.

Big Data Analysis

Processing massive astronomical datasets, developing algorithms for survey data, and preparing analysis tools for next-generation observatories like LSST.
